# My Blog

A personal blog and art-tech showcase — integrating **Bloom Filter technology** with a **retro-futuristic aesthetic**, inspired by *Steins;Gate*, microbial geometry, and jellyfish.

---
![](/public/static/images/logo.webp)
## 🧭 Overview

This site is more than a blog — it's an experimental platform where:

- You can **track user behavior** without invading privacy  
- Visuals evoke a **Nixie tube worldline meter**  
- The interface reflects **microbial forms and geometric elegance**  
- **Jellyfish-inspired** elements glow subtly in motion  

A fusion of **analytics, art, and identity**.

---

## 🔬 Bloom Filter Analytics (Private & Lightweight)

This blog uses **3 independent Bloom Filters** to track:

- **Visitors**: Identify unique/repeat visitors  
- **Page Views**: Trace content engagement  
- **Searches**: Monitor keyword interest  

🛡️ All data is **hashed and anonymized** — no raw user data is ever stored.

## 🧱 Tech Stack

- **Next.js 15** + **TypeScript**  
- **React Three Fiber** (WebGL 3D)  
- **Contentlayer v2** (MDX content)  
- **Tailwind CSS 4**  
- **Custom Bloom Filter library**  
- **Internationalization**: English 🇺🇸 / Japanese 🇯🇵
